In March of 2023, Goldman Sachs published a reportLinks to an external site., indicating that ~25% of the tasks in US and Europe can be automated using AI. However, as you can see in this visualizationLinks to an external site., not all industries will be affected equally. According to the report, certain jobs, like office tasks, legal, architecture and social sciences have a potential for 30%+ automation, while positions like construction, installation and building maintenance are going to be largely unaffected.

You can also see supporting evidence in the Facebook Research paperLinks to an external site., which highlights Moravec’s paradox, the thesis that the hardest problems in AI involve sensorimotor skills, not abstract thought or reasoning, which coincide with Goldman Sachs predictions.

While both of these papers are very impressive, they also heavily influenced by the recent advances in Large Language Models (LLMs). For this final project I have prepared a collection of ~200K news articles (about 900 MB) articles on our favorite topics: Data Science, Machine Learning and Artificial Intelligence, and I want you to identify what industries and job lines are going to be most impacted by the AI over the next several years, based on the information you can disseminate from this text corpus.

The objective of your final project is to identify what types of tasks and jobs are most likely to see the biggest impact from AI by extracting meaningful insights from unstructured text. Your goal is to provide actionable recommendations on what can be done with AI to automate the jobs and / or improve employee productivity. Please pay attention to the introduction of novel technologies and algorithms, such as AI for image generation and Conversational AI, as they represent the entire paradigm shift in adoption of AI technologies and data science in general.

def smart_join(tokens):
    text = ''
    for i, token in enumerate(tokens):
        if token in [".", ",", ";", ":", "!", "?"] and i > 0:
            text = text.rstrip() + token + " "
        else:
            text += token + " "
    return text.strip()

2. Minimize data cleaning for NER purpose¶

We carried out the first basic text cleansing, notably excluding the conversion to lowercase for NER.¶

Basic Principles in Text Cleaning for NER¶

  1. Case Sensitivity:
    • Standard Cleaning: Text is often converted to lowercase to maintain consistency and simplify tasks like topic modeling or word frequency analysis.
    • NER Cleaning: The case is retained since proper nouns, especially names of people and places, often start with capital letters. Converting to lowercase can reduce the accuracy of entity detection.
  2. Stopwords Removal:
    • Standard Cleaning: Stopwords are frequently removed to reduce dimensionality and noise.
    • NER Cleaning: It's often better to retain stopwords as removing them can disrupt entity phrases like "The United States of America."
  3. Tokenization:
    • Standard Cleaning: Text might be tokenized into individual words for analyses like word clouds or bag-of-words models.
    • NER Cleaning: Sentences should be kept intact or tokenized as whole sentences. Context around a word can be crucial for determining if it's an entity.
  4. Special Characters and Punctuation:
    • Standard Cleaning: These are often removed to simplify text.
    • NER Cleaning: Some special characters or punctuation might be retained, especially if they can denote entities (e.g., "$" for monetary values).
  5. Stemming and Lemmatization:
    • Standard Cleaning: Words are often stemmed or lemmatized to reduce them to their base or root form.
    • NER Cleaning: Avoid stemming and lemmatization, as entities need to be in their original form for accurate recognition.
  6. Context Preservation:
    • Standard Cleaning: Depending on the analysis, context might not always be critical.
    • NER Cleaning: Preserving the context is crucial, as surrounding words can influence the type and accuracy of entity recognition.
  7. Handling Numbers:
    • Standard Cleaning: Numbers might be removed, especially if the focus is on textual content.
    • NER Cleaning: Numbers should be preserved, especially if dates, amounts, or other numerical entities are relevant.

# Function to preprocess the news article
def clean_news_for_ner(text):
    # Remove HTML remnants, scripts, styles, comments, web links, and emails
    text = re.sub(html_remnants_pattern, '', text)
    text = re.sub(script_pattern, '', text)
    text = re.sub(style_pattern, '', text)
    text = re.sub(comment_pattern, '', text)
    text = re.sub(web_link_pattern, '', text)
    text = re.sub(email_pattern, '', text)

    # Remove newlines, tabs, and emojis
    text = text.replace('\n', ' ').replace('\r', '').replace('\t', ' ')
    text = re.sub(emoji_pattern, '', text)

    # Remove emojis and other special characters except alphanumeric, spaces, periods, and commas
    text = re.sub(r'[^\w\s.,]', ' ', text)

    # Normalize space (removing extra spaces)
    text = re.sub(r'\s+', ' ', text).strip()

    # Tokenize the text
    tokens = nltk.tokenize.word_tokenize(text)

    # Further cleaning with tokens
    tokens = [word for word in tokens if len(word) < 20]
    tokens = [word for word in tokens if not word.isnumeric()]

    # Join tokens back into a string
    # cleaned_text = ' '.join(tokens)
    cleaned_text = smart_join(tokens)

    return cleaned_text

# Function to preprocess text for sentiment analysis
def clean_news(text):
    
    # Tokenize the text
    tokens = nltk.tokenize.word_tokenize(text)
    
    # Apply length filters
    tokens = [word for word in tokens if len(word) > 1]

    # Filter out non-alphabetic tokens
    tokens = [word for word in tokens if word.isalpha()]
    
    # Lowercase the tokens for uniformity
    tokens = [word.lower() for word in tokens]

    # Remove stopwords
    tokens = [word for word in tokens if word not in stopwords]

    # Join tokens back into a string
    cleaned_text = ' '.join(tokens)

    return cleaned_text

# Function to determine the WordNet Part of Speech (PoS) tag for lemmatization
def get_wordnet_pos(word):
    """Map POS tag to first character lemmatize() accepts"""
    tag = nltk.pos_tag([word])[0][1][0].upper()
    tag_dict = {"J": wordnet.ADJ,
                "N": wordnet.NOUN,
                "V": wordnet.VERB,
                "R": wordnet.ADV}

    return tag_dict.get(tag, wordnet.NOUN)

# Function to lemmatize text (assuming it's already in lowercase)
def lemmatize_text(text):
    # Tokenize the text into words
    words = nltk.word_tokenize(text)
    
    # Initialize WordNetLemmatizer
    lemmatizer = WordNetLemmatizer()
    
    # Lemmatize each word using the appropriate part of speech
    lemmatized_words = [lemmatizer.lemmatize(word, get_wordnet_pos(word)) for word in words]
    
    # Join the lemmatized words back into a string
    lemmatized_text = ' '.join(lemmatized_words)
    
    return lemmatized_text

# Function to preprocess the news article
def clean_news_title_for_ner(text):
    # Remove parts of text after '-', '–', or '|' to remove media company's name
    text = re.sub(r'\s[-–|]\s.*$', '', text)

    # Remove HTML remnants, scripts, styles, comments, web links, and emails
    text = re.sub(html_remnants_pattern, '', text)
    text = re.sub(script_pattern, '', text)
    text = re.sub(style_pattern, '', text)
    text = re.sub(comment_pattern, '', text)
    text = re.sub(web_link_pattern, '', text)
    text = re.sub(email_pattern, '', text)

    # Remove newlines, tabs, and emojis
    text = text.replace('\n', ' ').replace('\r', '').replace('\t', ' ')
    text = re.sub(emoji_pattern, '', text)

    # Remove emojis and other special characters except alphanumeric, spaces, periods, and commas
    text = re.sub(r'[^\w\s.,]', ' ', text)

    # Normalize space (removing extra spaces)
    text = re.sub(r'\s+', ' ', text).strip()

    # Tokenize the text
    tokens = nltk.tokenize.word_tokenize(text)

    # Further cleaning with tokens
    tokens = [word for word in tokens if len(word) < 20]
    tokens = [word for word in tokens if not word.isnumeric()]

    # Join tokens back into a string
    # cleaned_text = ' '.join(tokens)
    cleaned_text = smart_join(tokens)

    return cleaned_text

# Function to check if text contains any of the specified keywords
def contains_keywords(text, keywords):
    # Create a regular expression pattern with all keywords, case insensitive
    pattern = re.compile('|'.join(re.escape(keyword) for keyword in keywords), re.IGNORECASE)
    # Search the text for any occurrence of the pattern
    return bool(pattern.search(text))

# Function to insert a line break after the second space
def add_line_break_after_second_space(label):
    words = label.split()
    if len(words) > 2:
        return ' '.join(words[:2]) + '\n' + ' '.join(words[2:])
    else:
        return label

# Summing up the counts by category
